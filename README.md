# AuditorIA-Risk-Compliance-Analytics

# Objetivo

Detectar riscos financeiros em transações corporativas através de análises automatizadas.

# Tecnologias
- Python </br>
- Pandas </br>
- Excel</br>
- Power BI</br>
- DAX</br>

# Principais análises
- Transações duplicadas </br>
- Valores acima de R$ 50.000 </br>
- Exposição financeira </br>
- Taxa de risco </br>
- Índice de conformidade </br>
- Ranking de fornecedores </br>

# Resultados
- 5.000 transações analisadas</br>
- 171 duplicidades detectadas</br>
- 217 transações de alto risco</br>
- 92,54% de conformidade
